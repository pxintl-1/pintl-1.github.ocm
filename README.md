# ʚ𝑳𝑶𝑽𝑬ɞ - 傳訊模擬互動平台 (含隨機樹洞功能)

這是一個針對二次元與夢角互動深度客製化的情感通訊模擬平台，並全新整合了微信風格的「隨機心聲樹洞」互動系統。

---

## 🌟 核心特色模組

- **獨立桌面 UI**：高度精美仿手機作業系統與主題更換功能。
- **同心商城**：模擬為夢角或自己挑選禮物、外賣並預訂送達。
- **TA的手機**：探索並查看與夢角相關的備忘錄與小秘密。
- **隨機心聲樹洞（全新功能）**：
  - **微信風外觀**：完美復刻經典的綠色發送泡泡、白色接收泡泡與對話小尖角。
  - **盲盒抽籤機制**：用戶可自由輸入文字投入公共字條庫，並可點擊按鈕隨機撈取其他用戶生成的心聲。
