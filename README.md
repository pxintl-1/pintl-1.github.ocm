# 🎲 隨機文字抽選樹洞應用程式 (Socket.io)

這是一個特別的隨機聊天互動系統。用戶可以自由上傳匿名文字（如笑話、心情、秘密）投入公共文字庫，並透過點擊按鈕，隨機從庫中抽取一條由其他在線（或曾連線）用戶所生成的內容。

## ⚙️ 運作邏輯
1. **上傳機制**：前端發送 `upload text` 事件 -> 後端接收並將字串 push 進伺服器記憶體陣列（`textPool`）。
2. **抽選機制**：前端發送 `request random` 事件 -> 後端利用 `Math.random()` 隨機抽樣 -> 單獨發送給該用戶，達成盲盒抽籤效果。

## 🛠️ 如何執行
1. 確保安裝 Node.js。
2. 執行 `npm install express socket.io`。
3. 執行 `node server.js`。
4. 打開 `http://localhost:3000` 即可開始體驗！
